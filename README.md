## 🌱 EcoRover Q
### Rover con Edge AI para monitoreo de áreas verdes

### 🚨 Problematica

Las ciudades enfrentan problemas como:

- Isla de calor urbana
- Contaminación
- Mantenimiento ineficiente de áreas verdes

Actualmente, el estado de las plantas se revisa de forma manual y tardía, lo que provoca:

- Desperdicio de agua
- Pérdida de vegetación

## How it Works

This example hosts a Web UI where we can see the video input from the camera connected via USB. The video stream is then processed using the `video_imageclassification` Brick. When a person is detected, it is logged along with the confidence score (e.g. 95% person).
